CMS - a content module - Manual

CMS is a small, light-weight contents handling module.

Keyfeatures

  • You can create contents.
  • You can upload images or photos in any sizes.
  • Set a image resize for the INDEX page, DETAIL page and RANDOM BLOCK.
  • Add the Sprockets Module to use the Tag and Category function.
  • Present your content with 4 integrated blocks.
  • A new art of comments with informations bubbles.
  • Guest find new comment link for login and register incl content redirect.
  • Set contents as archived, online and offline.
  • Use a free TextArea for your content history or for a new preview of content. ()It's visible for the admin only.)
-1-

First Start

There are one thing you should do before you try the new module. Please, follow the instructions.

Install the module "Sprockets"

At first we need to install the Sprockets Module. Important is to use version 2.0 or higher.

Create some tags with the Sprockets Module.

If you installed the Sprockets version >2.0, you can use into the cms module a category system too.

Check out the module preferences

There might be some preferences, you'd like to change before using the module first time. Go to Modules > CMS > Preferences

Choose the preferences like preferred. Think, most preferences should be explained enough. So I'll list just a few here, which can be important to explain:

  • You can display the contents with teaser text within the index page or as a table.
  • How many contents do you like to display in the index page? A good value is 3 or 5.
  • Breadcrumb: This is a small navigation on top of the page
  • Breadcrumb: This is a small navigation on top of the page
  • TOOLBAR: The Toolbar is visible within the content page only, not in the blocks. If you like you can enable a icon to print a page, create a PDF file, send a link via email, share the content.
  • Select box: If Sprockets are instelled, it's possible to display a select box of tags.
  • Freestyle: If you don't like to use the resize function, switch to the freestyle. After them, you can upload your images with your size.
  • Image file size: Please remember, the server has a limit too. In the most time you can not upload images higher 2MB.
  • Updated content: The updated content will be present on top. You can mark besides a content and display the notice "updated". In addition, you can select between 6 times to show this notice.
  • Comments: This module has some special effects to comment a content. First is, the bubble within the content incl comment counter. If you click as guest, you can not comment a content, but you can find two links. One is for login and one for thr registration. Both links using a redirect, back to the content. This is a new way in ImpressCMS.
  • Notification: This module can send notifications to the user if enabled. The user can select between two types if the administrator allows.

Done! Next step.

-2-

System settings

General Setting > Default Editors

Set the "TinyMCE" to the default editor.

General Setting > HTMLPurifier

If you like to add some special CSS and/or JS within your content, maybe the cms module would not show the right content. The reason is the HTMLPurifier, since he filtered some HTML tags. If you create a business website, maybe you should dissable it. But in the most case your should'nt dissable, because your security could be going down.

Groups > Webmaster

Edit the webmaster group and allow the WYSIWYG editor for the cms module. This means, the webmaster can use the TinyMCE.

Create a new content

Go to the Tab "Content" and click it. Enter a short but descriptive title on top for your new content. (Important is the following: If you create a content with the DHTMLeditor, dissable the option "Enable HTML tags". Otherwise the line break works not correct.)

Upload images or photos

Finally you can start uploading a image or photo for your content. Clicke the upload-button and select your file from your Computer or Digicam. Alternative you can choose a URL.

Content

The content is seperated in two parts. One is the teaser text, this will be display in the overview only with your uploaded image. The seconde is the main text and will be display within the detailpage from your content.

There is one more field for your History, ideas or reviews. You can put any content and this will not publish in the user page.

Others

The following fields, you don't need to fill out or to change: Enable HTML tags, Enable linebreak, Enable images, Enable smiley icons, Meta Keywords, Meta Description, Short URL.

PHP scripts

If you like to add a PHP script within your content, than it is not possible to add directly. It is very risky ans dangerous for your website. But ImpressCMS has a solution: Go to the "Custom Tags", create a new one and add your PHP script there. Now you can add your new custom add into your content.

-3-

Blocks

Currently you have 4 blocks for the 'CMS' module:

  • Random content
    • The "Random content" block will display (by default values) 5 contents with image an the update info, if available. You can change the numbers of contents if you like.
  • Ticker content
    • This block will display in a very small line the titles from your contents. If you don't put a block title, you get a very usefull and small ticker. You can change the nubers of contents if you like.
  • List content
    • This block display (by default values) the latest 3 titles in a list. You can change the numbers of contents also.
  • Slide content
    • This is a very nice slider. He will show the latest 3 contents (Please don't change the preferences in this block!). This block use by default a blue color. If you like to change, you need to touch the module CSS files or overwrite the class with your Theme.
  • Select content
    • A very useful block, since you can clone it. You can show the teaser text only or the full content in your block. Select your prefered content and setup to your position. Play with the "Block-Position" and "Symlinks" from ImpressCMS also.

And sure, you can clone all blocks, if needed and set your preferences by yourself.

-4-

Date in templates

Since CMS module version 2, the regular date was deleted, because you don't need within a content. But if you like, you can add the date in your templates with the following SMARTY codes.

  • <{$cms_start.date|date_format}>
    • Output "Feb 12, 2012"
  • <{$cms_start.date|date_format:"%Y/%m/%d"}>
    • Output "2012/02/12"
  • <{$cms_start.date}>
    • Output: "12.02.2012 09:40:00"
  • <{$cms_start.date|date_format:"%d.%m.%Y"}>
    • Output "12.02.2012"

More informatione here www.smarty.net - date format
